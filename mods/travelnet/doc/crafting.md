# Travelnet items
The Travelnet mod provides five craftable items:

## Travelnet box
Transports you to other boxes in the same network.

Recipe:
```
+-------+-------+-------+
| glass | steel | glass |
| glass | mese  | glass |
| glass | steel | glass |
+-------+-------+-------+
```

MineClone 2: Replace mese with a redstone block.


## Elevator
Transports you to eleveators on the same x and y coordinates.

Recipe:
```
+-------+-------+-------+
| steel | glass | steel |
| steel |       | steel |
| steel | glass | steel |
+-------+-------+-------+
```

MineClone 2: Use iron instead of steel ingot.


## Steel door
Recipe:
```
+-------+-------+-------+
| steel |       | steel |
| steel |       | steel |
| steel |       | steel |
+-------+-------+-------+
```

MineClone 2: Use iron instead of steel ingot.


## Glass door
Recipe:
```
+-------+-------+-------+
| glass |       | glass |
| glass |       | glass |
| glass |       | glass |
+-------+-------+-------+
```


## Tin door
Recipe:
```
+-------+-------+-------+
|  tin  |       |  tin  |
|  tin  |       |  tin  |
|  tin  |       |  tin  |
+-------+-------+-------+
```

Not available with MineClone 2.
